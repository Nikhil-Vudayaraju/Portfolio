# Personal Portfolio Website

## Overview

This is a responsive and modern personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio showcases my skills, certifications, projects, and contact information in a clean and interactive user interface.

## Features

* Responsive design for desktop and mobile devices
* Modern dark-themed UI inspired by professional developer portfolios
* Animated hero section
* Smooth scrolling navigation
* Scroll-based fade-in animations
* Interactive project showcase
* Certifications section
* Contact section with GitHub, LinkedIn, and Email links
* Resume download option

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
